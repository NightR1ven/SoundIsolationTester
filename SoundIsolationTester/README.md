# Sound Isolation Tester

Система тестирования звукоизоляции помещений с возможностью детекции спуфинг-атак.

## Возможности

- **Тестирование звукоизоляции**: Оценка эффективности звукоизоляции помещений
- **Мульти-движковое распознавание**: Поддержка Whisper, Vosk, Google Speech и других
- **Детекция спуфинг-атак**: Обнаружение попыток обмана системы тестирования
- **Подробная аналитика**: WER, CER, BLEU и другие метрики
- **Пакетное тестирование**: Массовое тестирование на датасетах
- **Генерация отчетов**: HTML, PDF, Excel, CSV форматы
- **Офлайн-работа**: Работа без интернета с локальными моделями
