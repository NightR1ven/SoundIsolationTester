# Sound Isolation Tester

Программа для тестирования звукоизоляции помещений с ИИ-анализом.

## Возможности
- Запись звука с двух микрофонов одновременно
- Анализ звукоизоляции с помощью локального ИИ
- Генерация отчетов в HTML и JSON форматах
- Визуализация результатов в реальном времени

## Установка
1. Скачайте папку с программой
2. Установите Python 3.8+
3. Запустите `pip install -r requirements.txt`
4. Запустите `python main.py`

## Использование
1. Подключите два микрофона
2. Выберите устройства в программе
3. Начните запись
4. Получите анализ звукоизоляции
